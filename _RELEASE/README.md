# GHZ Act 2 (ghz200) - Sandbox Stage

Replaces Green Hill Zone Act 2 (Modern) with a custom “sandbox” stage.
Current goal: load a simple custom terrain successfully (assets/objects later).

## Install (HedgeModManager)
1) Drop the mod folder (or the .zip) into your Mods directory.
2) Enable the mod in HedgeModManager.
3) Click Save and Play.

## What this mod changes
- Stage slot: ghz200 (Green Hill Act 2)
